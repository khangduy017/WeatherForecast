# INSTRUCTION

## Manually installation

-  Frontend
```shell
cd frontend
flutter pub get
```

-  Backend
```shell
cd backend
npm install
```

### Run

-  Frontend
```shell
cd frontend
flutter run
(Choose 'Chrome (chrome)')
```

-  Backend
```shell
cd backend
npm start
```
